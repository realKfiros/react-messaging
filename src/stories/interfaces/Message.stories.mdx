```typescript
export interface Message {

  text: string;

  date: Date;
  data?: any;
}
```

# Properties

| Name | Type   | Description                                |
| ---- | ------ | ------------------------------------------ |
| \_id | string | this identifier of the message             |
| text | string | this text of the message                   |
| user | User   | the sender of the message                  |
| date | Date   | this date that the message sent on         |
| data | any    | additional properties, for custom messages |
